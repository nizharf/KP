🚀 HOW TO USE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Navigate to:  /admin/transfers

2. Click button: "Scan QR untuk Pindah Aset"

3. Choose method:
   Option A: Click "Buka Kamera" → Scan QR code
   Option B: Type asset code → Press Enter

4. Fill form:
   - From Location: (auto-filled)
   - To Location: (your input)
   - Notes: (optional)

5. Submit:
   - Click "Proses Perpindahan"
   - Wait for success message
   - Page auto-refreshes
   - New entry in table


🔗 API ENDPOINTS

Lookup Asset:
  GET /admin/assets/by-code/{code}
  
  Response: {
    "success": true,
    "asset": {
      "id": 1,
      "asset_code": "AST-001",
      "name": "Laptop",
      "location": "Ruang IT",
      "category": "Electronics",
      "brand": "Dell",
      "model": "XPS"
    }
  }

Store Transfer:
  POST /admin/transfers
  Content-Type: application/json
  X-CSRF-TOKEN: {token}
  
  Request: {
    "asset_code": "AST-001",
    "from_location": "Ruang IT",
    "to_location": "Ruang Kepala",
    "transfer_date": "2026-01-20T10:30:00Z",
    "notes": "Optional notes"
  }
  
  Response: {
    "success": true,
    "message": "Perpindahan berhasil dicatat.",
    "redirect": "/admin/transfers"
  }
